## Getting Started

### Prerequisites
- [Node 8](https://nodejs.org/en/download/)
- [Yarn](https://yarnpkg.com/en/)
- [Serverless](https://serverless.com/framework/docs/getting-started/)

### Running the Application

Just run the following command:

> yarn start

### Running the Tests

> yarn test
